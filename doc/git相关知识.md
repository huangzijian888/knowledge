# @huangzijian

1.创建分支

	git branch 分支名

2.切换分支

	git checkout 分支名

以上两条语句可以合并为 git checkout -b 分支名

3.从远程仓库clone

	git clone git@github.com:用户名/pro名.git

4.从本地推送到远程仓库

	git push origin 分支名
	远程仓库默认为 origin

5.编辑README.md
	
	详情见前辈[博客](http://blog.csdn.net/u011068702/article/details/49741077)
